高宽、定位控制

width、height、position可以设定高宽和定位

position可以按如下方式定义:

  • 字符串,可以为top,left,right,bottom,center,middle,对齐屏幕。
  • 字符串,可以是rightnear,bottomnear,对齐点击元素。
  • 二元数组,数组元素可以为数字或字符串,第一个表示X轴方向的定位，第二个表示Y轴方向的定位。
  • 二元数组,数组元素第一个为rightnear或bottomnear,第二个随rightnear和bottomnear的不同分别表示X轴或Y轴的偏移量。
  • 三元数组,前两个数组元素同上，第三个随rightnear和bottomnear的不同分别表示Y轴和X轴的偏移量。

本地连接

如果content不为空，就会调用里面的字符串作为弹窗内容

如果为空，则会依次调用link属性、html代码中链接的href属性或按钮的alt属性

$('#exp_4').tip( { title:'Hello exp_4', content:'This is local content' } );
<button id='exp_4'>click me</button>

设定回执函数

run属性可以设定触发的事件,this为弹出窗口的HTMLElement对象

run可以用来从服务器传执行函数在客户端运行，也可以在客户端预设一些函数在窗口显示后运行

var options = { run: function(){ $(this).fadeOut('slow').slideDown().slideUp().show('slow').animate({left:0}); }}; $('#exp_8').tip(options);
<button id='exp_8' alt="#hello1">click me</button>
